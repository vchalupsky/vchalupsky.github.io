# Simple Blog (static)

This workspace contains a minimal static blog demo built with HTML, CSS and JavaScript.

Files added:
- [index.html](index.html)
- [css/styles.css](css/styles.css)
- [js/script.js](js/script.js)

Preview locally:

Windows (PowerShell or CMD):
```powershell
python -m http.server 8000
```
Then open http://localhost:8000 in your browser and view `index.html`.

Or simply open `index.html` directly in your browser.
